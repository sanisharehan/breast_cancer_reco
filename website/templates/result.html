Features involved in Breast Cancer Tissue Classification Using Non-CNN methods

Feature Description
Perimeter Number of nucleus boundary points
Area Number of points inside the nucleus
Compactness Calculated from perimeter and area of the nucleus using formula, C = 4pi * area/perimeter . This is the measure of the efficiency with which a boundary encloses an area.
Energy Measures the homogeneity in the segmented breast images. It is computed as the correlation between two gray levels a and b, located at unit-distance at an angle of 0 degree.
Contrast Provides local pixel intensity variation.
Entropy Statistical measure of randomness that can be used to characterize the texture of the input image.
Local Binary Patterns Texture descriptor which computes a local representation of texture. The representation is constructed by comparing each pixel with its surrounding neighborhood of pixels.
Parameter-Free Threshold Adjacency Statistics A morphological feature, with principle to accumulate in the histogram bins, pixels according to their number of white neighbors, in multiple-threshold binarized images.
ORB (Oriented FAST and Rotated BRIEF) Designed to be rotation invariant and noise tolerant, it provides FAST keypoint detectors and BRIEF keypoint descriptors.
